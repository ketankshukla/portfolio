My biggest challenge could be classified in two categories:

* My biggest challenge as a software developer
* My biggest challenge taking the Rails program at Bloc

As for the first one, it has always been learning the syntax of the language, especially when I'm learning a brand new language. Sometimes I wish that we only had one language for everything, that way we wouldn't have to worry about missing out those darn semi-colons at the end of statements in languages such as C#.

Luckily, with Ruby I don't have to worry about that, but Ruby comes with its own challenges where sometimes reading the code isn't as intuitive as one would think. I have encountered 6 lines of code compressed into one, but if I were to look at that one line of code without having seen the other six first, I would have never guessed that it was the result of a massive compression.

This is the one thing that still bothers me when dealing with the Ruby language - how far deep does this code go if I were to uncompress it? To be fair, I have to say that the compressed code does look elegant and simple compared to the more intuitive uncompressed code - however I've learnt that it might be a good idea to learn how that one-liner came to be, in case you see it next time around.

Now, as for my biggest challenge at Bloc, it has always been the race against time. I am in the 12 week crunched version of the Ruby on Rails course and I knew going in that I would have to work at least 40 hours a week to be able to keep up with the course. I found that it took me more than that because the 40 hours was just to learn the stuff. I then had to practice it, review the work, repeat it in different ways so I can finally incorporate it in my muscle memory (kind of like using the Vim editor).

Right now, I have completed the projects that I was required to do, almost finished my portfolio and about to start my Capstone project. I would have loved to finish the Blocipedia and Open Todo API projects, however I need to prioritize what's important first - and what is important is the Capstone and my Job Prep module. I am really looking forward to them.

In conclusion, challenges will come and go, but it's how I face them that will determine how efficiently I can handle them.
